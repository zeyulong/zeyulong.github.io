文档排版步骤

例如，当对参考文献列表中的文献条目进行插入、删除等操作后，引用编号不会随之更新。若要更新引用编号，则需进行以下操作：

  • 按 Ctrl+A 快捷键选中全文，然后按 F9 键即可更新引用编号。

  • 或者选中需要更新编号的部分，点击鼠标右键，选择更新域代码即可。

  1. 审阅原文档中设置在文中的样式。显示、更新正文、标题和题注样式。标题样式

  3. （参考文献处）插入和设置各处的分节符（方便不同页码格式）。

  4. 设置脚注、尾注。

  5. （文献编号样式）。

  6. 图片大小。

  7. 公式、图、表标号和引用使用“引用”选项卡中的“插入题注”和“交叉引用”。交叉引用

段落

在 “布局”-“页面设置”-右下角“更多”-“文档网格” 的“行数”栏中有一个“跨度”值，这个“跨度”就是1倍行距，行距的倍数就受到“跨度”的影响。例如，1.5倍行距就是“跨度”的1.5倍。因此，不同字体大小的行间距的1.5倍也是完全不同的。

A4:

  • 1倍行距——固定值15.6磅。
  • 1.5倍行距——固定值23.4磅。

标题样式

原文档设置过的样式会随着样式被修改而自动更新。如果牵一发而动全身的话，不妨新建样式，需要时点击应用。

  1. 开始-样式-翻动找到想要修改的样式-右键-修改。
    1. 对于没有找到的样式: 点击“更多”（点击样式栏右下角）打开“样式”窗口-点击底部的“管理样式”-推荐-翻动选中想要显示的样式-设置查看推荐的样式时是否显示该样式: 显示。
    2. 对于不希望显示在样式栏的样式，选择“使用前隐藏”。
    3. 同时，可以通过“设置按推荐的顺序排序时所采取的优先级”来调整显示在样式栏的顺序。
  2. 打开“修改样式”窗口后，修改: 样式基准、后续段落样式；点击左下角“格式”，修改字体、段落。
  3. 依次修改正文、各级标题和题注样式。其中，尤其是正文的样式，修改后可能会直接改变原文档的排版。因此建议新建样式，例如“正文-1”。然后在设置各级标题和题注样式时，在“后续段落样式”处选择“正文-1”。

多级标题

前期工作：设置好各级标题的样式。

  1. 开始-段落-多级列表-定义新的多级列表。

  2. 打开定义新多级列表窗口后，首先点击左下角的”更多“。

    1. 单击要修改的级别: 1。
    2. 将级别链接到样式: 标题 1
    3. 编号格式-此级别的编号样式。
    4. 编号格式-输入编号的格式。
    5. 位置（-设置所有级别）-编号对齐方式、对齐位置、文本缩进位置。
    6. 编号之后: 空格。
    7. 依次修改各级别。
    8. 确定。

  3. 后续希望继续修改当前的多级标题样式时，仍然是开始-段落-多级列表-定义新的多级列表。

一般的各级标题序号为: 1, 1.1, 1.1.1。因此，编号格式-此级别的编号样式: 1,2,3,...

对于序号要求为: 一, 1.1, 1.1.1，如果一级标题选择编号格式-此级别的编号样式: 一,二,三,...则后续的编号默认是一.1, 一.1.1。这种情况下，可以通过依旧设置一级标题选择编号格式-此级别的编号样式: 1,2,3,...然后在文档中在一级标题的后面手动加入”一、“，再选中一级标题前面的”1.“，右键-字体-效果-勾选”隐藏“，来达到”一, 1.1, 1.1.1“的效果。

交叉引用

前期工作：设置完“题注”的样式和多级标题。

交叉引用和文献标号绑定，只有对标号的操作才会导致更新的发生。即只对文献的增加和删除，更新阈操作才有效。其他操作会导致找不到引用源或者出现错误引用，如本来标号为[2]和[3]的参考文献，你只将[2]和[3]后面的内容进行交换，此时对文章进行更新阈，并不会导致文中的参考文献引用的变化。此处需要将[3]删除，然后再插入到[2]前面，或者其他类似的做法，这样可以使得更新阈操作有效。

  1. 引用-题注-插入题注-打开“题注”窗口。
  2. 如果没有想要的标签，可以点击选项-“新建标签”。
  3. 点击“编号”，选择想要的编号格式。（可能需要提前正确设置好多级标题样式）
  4. 插入题注后，在需要引用的地方，点击“引用”-题注-交叉引用。
  5. 在“交叉引用”窗口设置“引用类型”、“引用内容”，选择“引用哪一个题注”后，点击右下角“插入”。

分节符

如何给文档分节呢？只要在每一个要分节部分的第一个字的前面插入分节符即可。例如：我们给第一部分和第二部分分节，就把光标置于摘要前面，点击“布局”-“页面设置”-“分隔符”-“分节符”-“下一页”即可。

点击“开始”-“段落”-“显示/隐藏编辑标记”按钮，可显示或隐藏分节符。

尾注

运用插入尾注的方法来插入参考文献比较简单，适用于我们写期刊文献。在 Word 中可以通过双击标注到指定参考文献位置，或者双击参考文献自动定位到文中引用的地方，非常方便。

  • 基本操作：

    1. 打开Word，指定需要添加参考文献的地方，选择引用-插入尾注。

    2. 从知网上复制参考文献，并粘贴到所需插入的位置。

  • 设置尾注格式：

    • Word 默认情况下，在第一篇参考文献前面有一条短横线，如果参考文献跨页了，在跨页的地方还有一条长横线，这些线无法选中，也无法删除。这是尾注的标志，但一般科技论文格式中都不能有这样的线，删除办法如下：
      1. 切换到大纲视图：视图-视图-大纲。
      2. 切换到大纲视图后，引用-显示备注。
      3. 左下角选择“尾注分隔符”，这时那条短横线出现了，选中它，删除。
      4. 再在下拉菜单中选择“尾注延续分隔符”，这是那条长横线出现了，选中它，删除。
      5. 完成后，视图-视图-页面视图。横线消失了。
    • 删除尾注开头的空行。空行是因为删除了尾注横线后所留。
      1. 如上操作，视图-草稿；引用：显示备注；尾注：尾注分隔符 & 尾注延续分隔符-右键：段落-行距：固定值，0.7 磅（够小就行）。

  • 更改编号格式（i, ii -> 1, 2）：

    1. 引用-脚注-更多（点击脚注栏右下角）。
    2. “脚注和尾注”窗口-格式-编号格式-应用。

  • 给正文尾注左右添加中括号：

    1. 开始-编辑-替换-更多-特殊格式-尾注标记。也可以直接在“查找内容”处输入: ^e。
    2. 在“替换为”处输入: [^&]。

  • [1][2] 变 [1,2]

    1. 选中中间的][，右键-字体-效果-勾选: 隐藏。
    2. 在[12]的12中间添加一个,，后选中,-开始-字体-设置为上标。

  • 在任意位置处添加尾注（参考文献）：

    1. 引用-脚注-更多（点击脚注栏右下角）。
    2. “脚注和尾注”窗口-位置-尾注: 节的结尾-应用更改-将更改应用于: 本节-窗口右下角:应用。
    3. 自定义各节的结尾是否显示尾注（一般在参考文献节显示，其它节不显示）：
      1. 布局-页面设置-更多（点击页面设置栏右下角）。
      2. 页面设置窗口-布局-节-不勾选“取消尾注”。

在 Word 表格中构建文档

使用表格来构建整个文档在Word中确实会受到一些限制，比如插入图表和使用某些格式化功能。为了解决这个问题，可以考虑以下几种方法：

1. 脱离表格进行编辑

方法说明： - 将报告的主体内容从表格中剪切出来，直接在Word文档中以常规文本格式排版。 - 只在需要使用表格的特定部分使用表格，如数据对比、统计信息等。

操作步骤： 1. 选择表格中的内容。 2. 右键选择“剪切”或使用快捷键Ctrl+X。 3. 在Word文档中需要的位置点击右键选择“粘贴”或使用快捷键Ctrl+V。 4. 调整原表格的部分，只保留必须使用表格的内容。

或在正常文档中编辑内容，定稿后再按源格式粘贴进表格中。

2. 使用文本框或者形状

方法说明： - 如果某部分内容需要显著区分并需置于表格内，可以使用文本框或形状来插入这些内容。 - 文本框和形状可以自由移动且支持单独的格式化和样式，有助于绕过表格的限制。

操作步骤： 1. 在“插入”菜单中选择“文本框”或“形状”，根据需求选择合适的样式。 2. 在文档中合适的位置插入选中的文本框或形状。 3. 将需要特别格式化的内容输入文本框或形状中。

3. 分节符应用

方法说明： - 使用分节符可以在同一文档中应用多种不同的页面布局和格式化。 - 这使得部分内容可以脱离普通文本的限制进行特殊处理。

操作步骤： 1. 在需要开始新格式的位置插入分节符。 - 转到“布局”选项卡，点击“分隔符”选择“下一页分节符”。 2. 在新的节中，你可以自由更改页面布局（如横向页面）或进行复杂的格式化。

4. 利用Word样式和主题

方法说明： - 使用Word的内置样式和主题功能可以快速统一并美化文档的外观，同时不需要借助表格布局就能进行有效的内容组织。

操作步骤： 1. 选中需要应用样式的文字。 2. 在“开始”选项卡的“样式”中选择适合的样式应用。

选择合适的方法取决于你的具体需要与偏好。如果开题报告要求特别严格使用表格，考虑与指导教员或相关负责人沟通可能的格式调整。

Excel多属性筛选

  • 若你注重 便捷录入 和人眼可读性，且筛选统计需求普通，选择 方案一（单列，多属性用逗号分隔）。
  • 若你重视 后续精确分析、筛选、多属性统计与排序等高级功能，建议改为 方案二（多列法）。

方案一：属性多值合并于单元格，分隔符规范

推荐原因：

  • 便于阅读，不增加表格宽度
  • 满足多数“单属性和简单多属性”筛选需求

推荐分隔符：英文逗号 , 或 英文分号 ;，建议整个表格统一。以 , 为例。

示例

事物名 属性
A 属性1,属性2
B 属性2,属性3
C 属性1
D 属性1,属性2,属性3

操作方法：

  1. 单属性筛选。使用“文本筛选”功能，筛选包含某个词的行即可：

    • 筛选→文本筛选→包含…（输入“属性1”等）

  2. 多属性筛选（如“属性1+属性2”）。可以继续用“文本筛选”，选择“包含”两个条件，分别输入“属性1”和“属性2”，设为“并且”的关系即可。

  3. 排序

    • 按属性列排序，就是按属性字符串的首字母排序，不一定有实质意义。
    • 若要求复杂排序（如排序“车企”优先），可以新插一列，用公式如=IF(ISNUMBER(SEARCH("属性1",B2)),1,0)辅助排序。

  4. 分类统计与计数。可以配合COUNTIF、SEARCH等进行，示例：

    • =COUNTIF(B:B,"*属性1*") 统计含“属性1”的数量。

  5. 注意事项

    • 推荐属性词语之间 不要加空格，统一数据风格。
    • 最大劣势：不能很好处理严格的单个属性排序，也不适用于复杂的多属性联合分析。

  6. 完成。

“哑变量”（多列法/One-hot编码）

推荐原因

  • 利于严谨的属性统计、分类、排序和数据分析
  • 适合少量固定属性的情况

示例结构

事物名 属性1 属性2 属性3 属性4
A 1 0 1 0
B 0 1 0 1
C 0 0 0 1
D 1 1 1 0

填写时，“拥有该属性”则为1，否则为0。

文档排版步骤

例如，当对参考文献列表中的文献条目进行插入、删除等操作后，引用编号不会随之更新。若要更新引用编号，则需进行以下操作：

  • 按 Ctrl+A 快捷键选中全文，然后按 F9 键即可更新引用编号。

  • 或者选中需要更新编号的部分，点击鼠标右键，选择更新域代码即可。

要在 Word 中锁定域以防止编辑，可以按照以下步骤操作：

  • 锁定域：选中需要锁定的域，然后按下 “Ctrl+F11” 组合键，这样该域就被锁定了，无法被修改。
  • 解除锁定：如果需要解除域的锁定，可以单击该域，然后按下 “Ctrl+Shift+F11” 组合键。
  • 这种方法可以有效防止域内容被意外修改，确保文档的完整性。
  • 在转换为PDF时，锁定域可以避免图表等内容的自动更新。

段落、行距

在 “布局”-“页面设置”-右下角“更多”-“文档网格” 的“行数”栏中有一个“跨度”值，这个“跨度”就是1倍行距，行距的倍数就受到“跨度”的影响。例如，1.5倍行距就是“跨度”的1.5倍。因此，不同字体大小的行间距的1.5倍也是完全不同的。（这个说法仅供参考，未必绝对正确）

A4:

  • 1倍行距——固定值15.6磅。
  • 1.5倍行距——固定值23.4磅。

样式

原文档设置过的样式会随着样式被修改而自动更新。如果牵一发而动全身的话，不妨新建样式，需要时点击应用。

  1. 开始-样式-翻动找到想要修改的样式-右键-修改。
    1. 对于没有找到的样式: 点击“更多”（点击样式栏右下角）打开“样式”窗口-点击底部的“管理样式”-推荐-翻动选中想要显示的样式-设置查看推荐的样式时是否显示该样式: 显示。
    2. 对于不希望显示在样式栏的样式，选择“使用前隐藏”。
    3. 同时，可以通过“设置按推荐的顺序排序时所采取的优先级”来调整显示在样式栏的顺序。
  2. 打开“修改样式”窗口后，修改: 样式基准、后续段落样式；点击左下角“格式”，修改字体、段落。
  3. 依次修改正文、各级标题和题注样式。其中，尤其是正文的样式，修改后可能会直接改变原文档的排版。因此建议新建样式，例如“正文-1”。然后在设置各级标题和题注样式时，在“后续段落样式”处选择“正文-1”。

    5. 位置（-设置所有级别）-编号对齐方式、对齐位置、文本缩进位置。

      • 对齐位置 = 编号（数字/符号）本身 距离页面左边缘有多远。
      • 文本缩进 = 编号后面的文字 距离页面左边缘有多远。

    6. 编号之后: 空格。

    7. 依次修改各级别。

    8. 确定。

  3. 后续希望继续修改当前的多级标题样式时，仍然是开始-段落-多级列表-定义新的多级列表。

一般的各级标题序号为: 1, 1.1, 1.1.1。因此，编号格式-此级别的编号样式: 1,2,3,...

对于序号要求为: 一, 1.1, 1.1.1，如果一级标题选择编号格式-此级别的编号样式: 一,二,三,...则后续的编号默认是一.1, 一.1.1。这种情况下，可以通过依旧设置一级标题选择编号格式-此级别的编号样式: 1,2,3,...然后在文档中在一级标题的后面手动加入”一、“，再选中一级标题前面的”1.“，右键-字体-效果-勾选”隐藏“，来达到”一, 1.1, 1.1.1“的效果。

交叉引用

前期工作：设置完“题注”的样式和多级标题。

交叉引用和文献标号绑定，只有对标号的操作才会导致更新的发生。即只对文献的增加和删除，更新阈操作才有效。其他操作会导致找不到引用源或者出现错误引用，如本来标号为[2]和[3]的参考文献，你只将[2]和[3]后面的内容进行交换，此时对文章进行更新阈，并不会导致文中的参考文献引用的变化。此处需要将[3]删除，然后再插入到[2]前面，或者其他类似的做法，这样可以使得更新阈操作有效。

对于编号（序号）列表，如果想在已有编号（如编号7）前插入一个新编号（编号7变编号8），且保证被交叉引用的编号7会自动变成编号8，则应该在编号6结尾回车（或主动在编号6和原编号7之间插入空行和点击“编号”以插入新编号）以生成新的编号7，且编号7自动变成编号8。不能在编号7开头回车以把原来编号7的内容“挤到”编号8去，这样做的话交叉引用的还会是编号7而不会变成更新后的编号8。

为什么会出现这样的情况，我想大概是与word的编号逻辑有关。

  1. 在编号6结尾回车以生成新的编号7（或主动在编号6和原编号7之间插入空行和点击“编号”以插入新编号），则原编号7及其交叉引用的“标签”都会随之变化。也就是必须保证原编号的“标签”自动更新了。
  2. 在编号7开头回车以把原来编号7的内容“挤到”编号8去，则编号7的“标签”并没有变，那么交叉引用的“标签”就不会变。

上述问题在题注为表题时换汤不换药。因此，想在表题前插入一行，不要在表题所在行首按下回车键，这会导致表题的域名被更新为新建行的内容。而要在表题上方一行的末尾按下回车，这样更新域后没有问题。

  1. 引用-题注-插入题注-打开“题注”窗口。
  2. 如果没有想要的标签，可以点击选项-“新建标签”。
  3. 点击“编号”，选择想要的编号格式。（可能需要提前正确设置好多级标题样式）
  4. 插入题注后，在需要引用的地方，点击“引用”-题注-交叉引用。
  5. 在“交叉引用”窗口设置“引用类型”、“引用内容”，选择“引用哪一个题注”后，点击右下角“插入”。

分节符

如何给文档分节呢？只要在每一个要分节部分的第一个字的前面插入分节符即可。例如：我们给第一部分和第二部分分节，就把光标置于摘要前面，点击“布局”-“页面设置”-“分隔符”-“分节符”-“下一页”即可。

点击“开始”-“段落”-“显示/隐藏编辑标记”按钮，可显示或隐藏分节符。

尾注

运用插入尾注的方法来插入参考文献比较简单，适用于我们写期刊文献。在 Word 中可以通过双击标注到指定参考文献位置，或者双击参考文献自动定位到文中引用的地方，非常方便。

  • 基本操作：

    1. 打开Word，指定需要添加参考文献的地方，选择引用-插入尾注。

    2. 从知网上复制参考文献，并粘贴到所需插入的位置。

  • 设置尾注格式：

    • Word 默认情况下，在第一篇参考文献前面有一条短横线，如果参考文献跨页了，在跨页的地方还有一条长横线，这些线无法选中，也无法删除。这是尾注的标志，但一般科技论文格式中都不能有这样的线，删除办法如下：
      1. 切换到大纲视图：视图-视图-大纲。
      2. 切换到大纲视图后，引用-显示备注。
      3. 左下角选择“尾注分隔符”，这时那条短横线出现了，选中它，删除。
      4. 再在下拉菜单中选择“尾注延续分隔符”，这是那条长横线出现了，选中它，删除。
      5. 完成后，视图-视图-页面视图。横线消失了。
    • 删除尾注开头的空行。空行是因为删除了尾注横线后所留。
      1. 如上操作，视图-草稿；引用：显示备注；尾注：尾注分隔符 & 尾注延续分隔符-右键：段落-行距：固定值，0.7 磅（够小就行）。

  • 更改编号格式（i, ii -> 1, 2）：

    1. 引用-脚注-更多（点击脚注栏右下角）。
    2. “脚注和尾注”窗口-格式-编号格式-应用。

  • 给正文尾注左右添加中括号：

    1. 开始-编辑-替换-更多-特殊格式-尾注标记。也可以直接在“查找内容”处输入: ^e。
    2. 在“替换为”处输入: [^&]。

  • [1][2] 变 [1,2]

    1. 选中中间的][，右键-字体-效果-勾选: 隐藏。
    2. 在[12]的12中间添加一个,，后选中,-开始-字体-设置为上标。

  • 在任意位置处添加尾注（参考文献）：

    1. 引用-脚注-更多（点击脚注栏右下角）。
    2. “脚注和尾注”窗口-位置-尾注: 节的结尾-应用更改-将更改应用于: 本节-窗口右下角:应用。
    3. 自定义各节的结尾是否显示尾注（一般在参考文献节显示，其它节不显示）：
      1. 布局-页面设置-更多（点击页面设置栏右下角）。
      2. 页面设置窗口-布局-节-不勾选“取消尾注”。

查找引用

  1. 开始-编辑-查找-高级查找：

  2. 更多-勾选“使用通配符”。

  3. 在 “查找内容” 框中输入以下代码：

    更宽松的查找：\[*\]

宏

录制宏

录制宏时，能使用快捷键操作就使用快捷键，而不是鼠标点击和选择。

  1. 点击 Word 顶部菜单栏的 “视图” (View) > “宏” (Macros) > “录制宏” (Record Macro)。
  2. 宏名：输入一个名字，例如 FirstLineIndent2（或者中文 首行缩进2字符）。
  3. 将宏保存在：选择 “所有文档 (Normal.dotm)”（这样你在任何 Word 文档里都能用）。
  4. 点击 “确定” 开始录制（此时鼠标指针旁会出现一个小磁带图标）。
  5. 执行操作：
    • 在任意空白处右键点击，选择 “段落” (Paragraph)。
    • 在“缩进和间距”选项卡下，找到 “特殊格式” (Special)。
    • 选择 “首行缩进” (First line)。
    • 在后面的磅值/字符值框中，确保显示为 “2 字符”（如果是数字如 14.9 磅，手动改为 2 字符 或 2 ch）。
    • 点击 “确定” 关闭段落对话框。
  6. 停止录制：
    • 点击底部状态栏的 “停止录制” 方块，或者去 “视图” > “宏” > “停止录制”。

容器功能与组合功能的区别

特性 容器 (Container) 组合 (Group)
核心关系 包含关系 (Parent-Child) 强调“内部”与“外部”的空间归属。 集合关系 (Collection) 强调将多个独立对象捆绑成一个整体。
大小调整 动态/自动 随内部内容的增减或移动自动扩大/缩小。 静态/手动 内部内容移出边界，组合框不会自动变大；需手动拉伸。
添加内容 智能吸附 将形状拖入容器区域，自动成为其“子元素”。 无逻辑关联 形状只是视觉上在框内，逻辑上并未“进入”组合，除非重新执行组合命令。
标题栏 原生支持 可直接在容器属性中添加/编辑标题。 不支持 需手动添加文本框并组合进去。
视觉样式 通常有专门的背景填充和边框样式，强调区域感。 只是多个形状的集合，默认没有统一的背景（除非特意画个矩形再组合）。
删除行为 删除容器本身，可以选择保留内部形状。 删除组合，通常会解散组合或连同内部形状一起删除（取决于操作）。
适用场景 架构图中的子系统、泳道、逻辑分区、云环境范围。 一个由多个零件组成的单一设备（如一台服务器由机箱+屏幕组成）、固定的图标集合。

如果选错了绘图类型，怎么改？

方法一：直接添加缺失的形状库（最推荐，保留当前内容）

如果你只是发现左侧缺少某些图标（例如选了“基本流程图”但突然想画个“数据库”），不需要换模板，直接加形状即可。

  1. 点击顶部菜单的 更多形状 (More Shapes)。
  2. 在弹出的菜单中，你会看到按类别分类的形状库（如“网络”、“工程”、“商务”等）。
  3. 找到你需要的类别，点击对应的子菜单（例如 网络 > 基本网络形状）。
  4. 该形状库会立即出现在左侧边栏中，你可以直接拖拽使用。
    • 技巧：你可以同时打开十几个不同的形状库，它们会堆叠在左侧，互不干扰。

方法二：更换当前文件的模板基础（彻底切换风格）

如果你想把整个文件的默认设置（如背景、主题、默认线条风格）完全换成另一种模板的风格：

  1. 点击左上角的 文件 (File)。
  2. 点击 新建 (New)。
  3. 此时你会看到模板选择界面。不要关闭当前文件。
  4. 选择你想要的那个正确模板（例如“跨职能流程图”）。
  5. 点击 创建 (Create) 。
    • 注意：这会打开一个新窗口（新文件）。
  6. 回到旧文件，按 Ctrl+A 全选所有图形，Ctrl+C 复制。
  7. 切换到新文件，Ctrl+V 粘贴。
    • Visio 通常能很好地保留格式。如果某些形状在新模板中不存在，Visio 会保留它们，只是可能失去一些智能连接特性。
  8. 保存新文件，删除旧文件即可。

方法三：调整页面设置（如果是尺寸或方向错了）

如果你选的模板是对的，但是页面大小（如 A4 变成了 Letter）或方向（横向变成了纵向）不对：

  1. 点击顶部菜单的 设计 (Design) 选项卡。
  2. 在 页面设置 (Page Setup) 区域：
    • 点击 大小 (Size) 修改纸张尺寸（可以直接输入厘米数，适合论文）。
    • 点击 方向 (Orientation) 切换横向/纵向。
    • 点击 背景 (Background) 修改背景。
  3. 这些设置可以随时无限次修改，不影响已画好的图形。

在表格中构建文档

使用表格来构建整个文档在Word中确实会受到一些限制，比如插入图表和使用某些格式化功能。为了解决这个问题，可以考虑以下几种方法：

1. 脱离表格进行编辑

方法说明： - 将报告的主体内容从表格中剪切出来，直接在Word文档中以常规文本格式排版。 - 只在需要使用表格的特定部分使用表格，如数据对比、统计信息等。

操作步骤： 1. 选择表格中的内容。 2. 右键选择“剪切”或使用快捷键Ctrl+X。 3. 在Word文档中需要的位置点击右键选择“粘贴”或使用快捷键Ctrl+V。 4. 调整原表格的部分，只保留必须使用表格的内容。

或在正常文档中编辑内容，定稿后再按源格式粘贴进表格中。

2. 使用文本框或者形状

方法说明： - 如果某部分内容需要显著区分并需置于表格内，可以使用文本框或形状来插入这些内容。 - 文本框和形状可以自由移动且支持单独的格式化和样式，有助于绕过表格的限制。

操作步骤： 1. 在“插入”菜单中选择“文本框”或“形状”，根据需求选择合适的样式。 2. 在文档中合适的位置插入选中的文本框或形状。 3. 将需要特别格式化的内容输入文本框或形状中。

3. 分节符应用

方法说明： - 使用分节符可以在同一文档中应用多种不同的页面布局和格式化。 - 这使得部分内容可以脱离普通文本的限制进行特殊处理。

操作步骤： 1. 在需要开始新格式的位置插入分节符。 - 转到“布局”选项卡，点击“分隔符”选择“下一页分节符”。 2. 在新的节中，你可以自由更改页面布局（如横向页面）或进行复杂的格式化。

4. 利用Word样式和主题

方法说明： - 使用Word的内置样式和主题功能可以快速统一并美化文档的外观，同时不需要借助表格布局就能进行有效的内容组织。

操作步骤： 1. 选中需要应用样式的文字。 2. 在“开始”选项卡的“样式”中选择适合的样式应用。

选择合适的方法取决于你的具体需要与偏好。如果开题报告要求特别严格使用表格，考虑与指导教员或相关负责人沟通可能的格式调整。

  3. 排序

    • 按属性列排序，就是按属性字符串的首字母排序，不一定有实质意义。
    • 若要求复杂排序（如排序“车企”优先），可以新插一列，用公式如=IF(ISNUMBER(SEARCH("属性1",B2)),1,0)辅助排序。

  4. 分类统计与计数。可以配合COUNTIF、SEARCH等进行，示例：

    • =COUNTIF(B:B,"*属性1*") 统计含“属性1”的数量。

  5. 注意事项

    • 推荐属性词语之间 不要加空格，统一数据风格。
    • 最大劣势：不能很好处理严格的单个属性排序，也不适用于复杂的多属性联合分析。

  6. 完成。

“哑变量”（多列法/One-hot编码）

推荐原因
  • 利于严谨的属性统计、分类、排序和数据分析
  • 适合少量固定属性的情况
示例结构
事物名 属性1 属性2 属性3 属性4
A 1 0 1 0
B 0 1 0 1
C 0 0 0 1
D 1 1 1 0

填写时，“拥有该属性”则为1，否则为0。

  1. 首先打开PPT文档，然后点击上方菜单栏的“视图”菜单，接着点击“幻灯片母版”选项。

  • 鼠标点一下PPT里的备注栏，再选上面 视图，缩放（放大镜），选择100%，字体就变大了。

更改幻灯片大小

  1. 打开你的PPT文件，点击顶部菜单栏的 “设计” 选项卡。
  2. 在工具栏最右侧找到并点击 “幻灯片大小” 按钮。
  3. 在下拉菜单中，直接选择 “宽屏 (16:9)”。
  4. 【最关键一步】此时PPT会弹出一个提示框，询问你如何处理原有内容，务必选择 “确保适合”。
    • 确保适合：系统会自动等比缩小原有内容，保证所有文字、图片都完整保留在16:9的画布内，上下可能会留出空白，但绝对不会裁切掉任何内容。
    • 最大化：系统会强行拉伸内容去填满新画布。从4:3转16:9时选这个，左右两侧的内容极大概率会被“切掉”，导致排版错乱。
  5. 完成。
  • 检查母版：如果每一页都有Logo或页脚跑偏了，可以进入 “视图” -> “幻灯片母版”，统一调整这些固定元素在16:9画布中的位置，这样所有页面会自动同步更新。

      可以同时勾选行高相等，以解决多行上下有空白的问题。

  4. 完成。
